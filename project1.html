FaceCard - 얼굴로 명함을 읽다

사용자의 얼굴을 실시간으로 인식하고 해당 인물의 명함 정보를 자동으로 불러오는 시스템

Keywords
Face Recognition Business Card OCR

프로젝트 소개

“ 사람 얼굴만으로 명함 정보를 확인할 수 있다면 어떨까?”

이 물음에서 출발한 FaceCard는 얼굴 인식 기술과 명함 OCR을 결합해, 사용자의 얼굴을 실시간으로 인식하고 해당 인물의 명함 정보를 자동으로 불러오는 시스템입니다.

해당 프로젝트는 AI학부 전공수업인 컴퓨터 비전 수업 팀 프로젝트로, OpenCV를 중심으로 다양한 오픈소스 라이브러리를 활용해 개발했습니다. 단순한 기술 구현을 넘어서, 얼굴이라는 자연스러운 입력을 통해 디지털 명함 정보를 연결하는 실시간 개인 식별 도구를 만들고자 했습니다.

주요 기능

명함 OCR 기능: 자동 투시 변환을 거쳐 명함 이미지를 저장 후, 텍스트를 자동 추출해 저장

얼굴 등록 및 인코딩: 사용자의 얼굴 이미지 저장 및 고유 벡터값 생성

실시간 얼굴 인식: 카메라에 인물이 등장하면 자동으로 정보를 매칭

정보 표시: 인식된 사람의 이름,이메일,회사명을 화면에 시각적으로 표시

개인 태그 색상 지정: 등록된 사람마다 원하는 색상 지정 가능

기술 스택 및 구현 방식

OpenCV: 이 프로젝트의 전반적인 라이브러리, 얼굴 인식 및 실시간 카메라 프레임 처리,자동 투시 변환

Pytesseract: 명함 이미지에서 텍스트 추출을 위한 OCR

PIL, Numpy: 이미지 전처리 및 연산

Pygame: 실시간 정보 디스플레이 UI 구현

JSON, Regex: 사용자 정보 저장 및 텍스트 정제 처리

얼굴 인식은 face_recognition 기반 라이브러리를 참고해 구현했고, 명함 자동 투시 변환의 경우 소벨, 모폴로지 등 이미지 전처리 단계를 통해 정확도를 개선했습니다.

프레임워크

역할

해당 프로젝트는 2인 팀프로젝트로 진행되었습니다. 저는 다음과 같은 파트를 중심으로 맡았습니다.

- 시스템 전체 설계: 얼굴 → 명함 정보로 이어지는 데이터 흐름 설계

- 얼굴 인식 및 정보 매칭 로직 개발

- OCR 성능 개선을 위한 이미지 전처리 튜닝

- 최종 시연 데모 구현 및 결과 정리

- PPT 발표
