프로젝트 소개

Botory(보토리)는 개인의 독서 기록을 쉽게 작성하고 관리할 수 있는 프로그램입니다.

이름은 “Book”과 “Story”의 합성어로, 한글로 발음하면 도토리를 연상되게 합니다. 작은 도토리를 하나하나 모으듯 책 한 권, 감상 하나를 차곡차곡 쌓아가는 감성적 경험을 의도했습니다. 이를 반영해 도토리 캐릭터를 마스코트로 기획하여 사용자에게 친근감을 더했습니다.

본 프로젝트는 Java 프로그래밍 전공 수업의 과제로 개발되었으며, 현재는 로컬 환경에서 실행되는 개인용 독서 기록 관리 프로그램 입니다.

주요 기능

사용자 인증: 회원가입 및 로그인 기능 제공

도서 검색: Kakao book API 연동으로 원하는 책 검색 가능

독서 기록 저장: 감상문, 읽은 날짜 등 자유로운 기록

기록 대시보드: 내가 쓴 모든 독서 기록을 한 눈에 확인 가능

도토리 캐릭터 기획: 사용자에게 친화적 UI요소 활용

기술 스택 및 구현 방식

• Java23, Spring Boot

• HTML, Thymeleaf

• DB - H2(인메모리 기반)

• Kakao API

• IntelliJ IDEA

현재는 테스트 목적의 H2 데이터베이스를 사용하지만, 추후 영구 저장소로 교체할 예정입니다.
