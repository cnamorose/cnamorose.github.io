InventoryManager – 로컬 기반 재고 관리 시스템

문구점을 테마로 한 소규모 매장 관리 시나리오

Keywords
Inventory Management LocalSystem

프로젝트 소개

InventoryManager는 C#과 Windows Forms를 활용해 구현한 로컬 환경 전용 재고 관리 프로그램으로, 사용자와 관리자의 역할을 분리해 재고·판매·사용자 정보를 효율적으로 관리할 수 있도록 설계한 데스크톱 애플리케이션입니다.

본 프로젝트는 고급 객체지향 프로그래밍 과목의 중간 과제로 개발되었으며, 객체지향 설계 원칙을 실제 프로그램 구조에 적용하고 GUI 기반 애플리케이션의 전체 흐름을 직접 설계·구현하는 데 초점을 두었습니다.

주요 기능

사용자 / 관리자 역할 분리: 관리자(Admin)와 일반 사용자(User)에 따라 접근 가능한 기능을 구분

사용자 인증 기능: 회원가입 및 로그인 기능 제공

재고 관리 (관리자 전용): 상품 등록, 수정, 삭제, 입고/출고 처리, 판매 내역 확인

상품 구매 기능 (사용자): 상품 조회, 장바구니 담기 및 결제 진행

데이터 저장 방식

본 프로젝트는 데이터베이스를 사용하지 않고, XML 파일 기반 데이터 저장 방식을 채택했습니다.

- Products.xml, Users.xml, StockIn.xml, Sales.xml 등으로 데이터 관리

- 프로그램 종료 후에도 데이터가 유지되도록 설계

- DB 없이도 데이터 영속성을 확보하는 구조 구현

이후 확장 가능성을 고려해 MySQL / SQLite 등 관계형 데이터베이스로의 전환을 염두에 둔 구조로 설계했습니다.

기술 스택 및 구현 방식

Language: C#

Framework: .NET Framework (Windows Forms)

Data Storage: XML

IDE: Visual Studio 2022

OS: Windows
